**Informations de fonctionnement sur KT-Drop**

Aperçu des fichiers ;
* œil = aperçu
* flèche = téléchargement
* poubelle = suppression

L’aperçu fonctionne pour :

* images
* PDF
* fichiers texte (txt, log, csv, json, xml, md, etc.)

Le reste affiche un message propre avec un bouton de téléchargement.
